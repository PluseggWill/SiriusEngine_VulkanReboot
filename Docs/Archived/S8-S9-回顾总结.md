## 一句话

`S8` 分支把 HybridDeferred 从「**有 DDGI 原型 + 裸镜面 IBL**」推进到「**可选 DDGI 探针 GI、行业对齐的间接镜面栈（prefilter mips + specular occlusion + Hi-Z SSR）、Vulkan ZO 阴影、共用 Temporal 合同（Halton + G-buffer MV + TAA v0.5）**」，并关闭 **G5**；路线图收口后队列指向 **S10 内容管线**。

---

## 🎯 要解决什么问题？

| 阶段 | 痛点 | 后果 |
|------|------|------|
| **S8** | 无全局间接漫反射层；AO 无时域稳定 | Stage 3 / 大场景「空」感；AO 噪声影响后续 GI |
| **盲修区间** | 阴影 Z 约定混 OpenGL/Vulkan；IBL 能量与 AO×镜面耦合混乱 | 漏光、过曝、暗角、玻璃感 |
| **Specular stack** | 仅 distant cubemap、无 prefilter mips、无 specular occlusion、无 SSR | 屋檐/拐角「漏天空」；粗糙反射全是锐利天空 |
| **S9** | 无统一 MV / 抖动 / history 生命周期；TAA 缺位 | 运动会闪烁；AO/SSR 各自一套 prev 矩阵 |

**目标形态：**

```text
S8：Temporal AO 基线 → DDGI probe update + deferred 采样（可关）
Lighting 修缮：shadow ZO、diffuse IBL π、sun intensity、GTAO 默认等
Specular stack：GGX prefilter → Lagarde SO → Hi-Z SSR（temporal lit HDR）→ C v0 cones/local box
S9 / G5：Vk_TemporalState + G-buffer MV + TAA v0.5；AO/SSR 共享 lifetime
```

### 3️⃣ Specular IBL 栈（[`specular-ibl-stack_Plan.md`](plans/specular-ibl-stack_Plan.md)）

| Phase | 交付 |
|-------|------|
| **A** | Offline GGX prefilter mips；`Pbr_SpecularOcclusion`（Lagarde）；deferred 镜面不再用裸 AO×IBL |
| **B** | Hi-Z `SsrTrace`；deferred `mix(prefilter, SSR, confidence) * specularOcc` |
| **B+** | SSR 采样 **上一帧 lit HDR**（`RecordHistoryUpdate`）；无效历史 → albedo |
| **C v0** | GTAO bent-normal cone SO（开关）；ImGui 局部 box probe（parallax，DDGI 关时） |

**间接镜面稳态公式（已落地）：**

```text
specularOcc = SpecularOcclusion(NdotV, ao|cones, roughness)
specularEnv = PrefilterSplitSum(R, roughness)  // ± local box
specularIndirect = mix(specularEnv, ssrRgb, confidence) * specularOcc
```
